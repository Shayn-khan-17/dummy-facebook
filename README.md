# Dummy Facebook

A social media clone application where users can login, signup, send friend requests, create posts, and see friends' posts.

## Features

- User Signup and Login
- Create and upload text posts
- Send friend requests to other users
- Accept or decline friend requests
- View posts from friends only
- Profile page for each user

## How to Use

1. Sign up with your email and password
2. Login to your account
3. Search for other users and send friend requests
4. Create posts on your timeline
5. Accept friend requests from others
6. View friends' posts on your feed
